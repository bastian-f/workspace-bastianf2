Instrucciones

  1. Si nuestro sistema operativo es Windows y no tenemos instalados los controladores USB para nuestro dispositivo K-PROX, deberemos descargar el siguiente archivo ZIP: Descarga

    Una vez descargado y descomprimido, ejecutaremos el fichero .exe correspondiente a nuestra plataforma.

  2. Descargar y ejecutar el controlador de puerto serie del menú desplegable situado arriba a la derecha para establecer una conexión segura con el servidor.

  3. Conectamos el K-PROX al ordenador mediante un cable USB y actualizamos la lista de dispositivos ejecutando el siguiente enlace: Recargar

  4. Seleccionar el dispositivo K-PROX de la lista de dispositivos.

  5. Actualizamos la última versión del firmware para K-PROX pulsado el botón 'Reprogram'.
